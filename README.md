Description:
SnackStreet is a modern food-themed website designed using only HTML and CSS. The purpose of this project is to demonstrate how a visually appealing and user-friendly food webpage can be built without using any JavaScript or backend technologies.

The website layout showcases attractive food sections, restaurant branding, menu highlights, and call-to-action elements commonly found in professional food service websites. It focuses on clean structure, responsive design elements, and creative use of colors, fonts, and images to represent a real-world food brand experience.

Key Features:

Built with pure HTML and CSS

Creative and professional food-themed UI

Responsive layout structure

Sections for menu items, featured dishes, and brand presentation

Eye-catching design with engaging visuals

SnackStreet demonstrates how strong visual design and organized structure alone can create a high-quality web interface for food lovers and businesses.
